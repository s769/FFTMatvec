Perform matrix-vector multiplication.

This function performs matrix-vector multiplication using the provided matrix and vectors.

Parameters:

  • out_vec – Pointer to the output vector.

  • in_vec – Pointer to the input vector.

  • mat_freq_tosi – Pointer to the matrix in frequency domain (transpose of input, scaled).

  • block_size – The block size for matrix operations.

  • num_cols – The number of columns in the matrix.

  • num_rows – The number of rows in the matrix.

  • conjugate – Flag indicating whether to perform conjugate multiplication.

  • full – Flag indicating whether to perform full matrix-vector multiplication.

  • device – The device ID.

  • scale – The scaling factor.

  • nccl_row_comm – The NCCL communicator for row-wise communication.

  • nccl_col_comm – The NCCL communicator for column-wise communication.

  • s – The CUDA stream.

  • in_vec_pad – Pointer to the padded input vector.

  • forward_plan – The forward FFT plan.

  • inverse_plan – The inverse FFT plan.

  • forward_plan_conj – The forward FFT plan for conjugate multiplication.

  • inverse_plan_conj – The inverse FFT plan for conjugate multiplication.

  • out_vec_pad – Pointer to the padded output vector.

  • in_vec_freq – Pointer to the input vector in frequency domain.

  • out_vec_freq_tosi – Pointer to the output vector in frequency domain (transpose of input, scaled).

  • in_vec_freq_tosi – Pointer to the input vector in frequency domain (transpose of input, scaled).

  • out_vec_freq – Pointer to the output vector in frequency domain.

  • cublasHandle – The handle for the cuBLAS library.

  • mat_freq_tosi_other – Pointer to the matrix in frequency domain (transpose of input, scaled) on other devices.

  • res_pad – Pointer to the padded result vector.
