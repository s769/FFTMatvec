Represents a matrix and provides matrix operations.

Parameters:

  • comm – The communication object.

  • num_cols – The number of columns in the matrix.

  • num_rows – The number of rows in the matrix.

  • block_size – The block size of the matrix.

  • conjugate – Flag indicating if the matrix is conjugate.

  • full – Flag indicating if the matrix is full.
